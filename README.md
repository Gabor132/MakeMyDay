## Getting Started

### Prerequisites

* Maven
* JDK 8
* Docker

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Gabor132/MakeMyDay.git
   ```
2. Run the docker/docker-compose file to startup the local DB
3. Run ./useful_scripts/run
4. Access application via browser
